Promises

Las promises se utilizan para escribir código asincrónico, es decir, tareas que no tienen un resultado inmediato. Cuando hacés un pedido asincrónico a una API tenés dos posibilidades:
1 - Que salga todo bien "resolve".
2 - Que algo salga mal "reject".
