Representacion de strings

JavaScript nos ofrece diferentes modos de representar strings, pero hay que tener cuidado con el tipo de comillas que utilizamos. Veamos el siguiente ejemplo en donde vamos a definir dos variables nombre y posición y le asignamos a cada una un string:

El el primer caso se concatenó las variables nombre y posicion con el string "juega de" utilizando el signo "+" y se muertra correctamente por consola.

En el segundo caso llamamos a las variables utilizando ${ } pero notamos que no salio como esperabamos; JavaScript nos convirtio todo a string en lugar de tomar los valores de las variables. Esto se debe a que utilizamos las comillas dobles ("..").

En el Tercer caso y solucionando el problema del caso anterior utilizamos las comillas simples ('..') y vemos que JavaScript en este caso nos trajo los valores como esperabamos.
