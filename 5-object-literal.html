Object Literal

Un object literal de JavaScript es una lista separada por comas de pares de nombre-valor (nombre: valor) envueltos en llaves. Los literales de objeto encapsulan los datos, encerrándolos en un paquete ordenado. Esto minimiza el uso de variables globales que pueden causar problemas al combinar código.
